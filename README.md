# Homestay Booking & Management System

Project Overview
This is my Final Year Project (FYP)—a secure web-based platform designed to streamline homestay bookings and enhance guest experience through an AI-powered chatbot.

Core Features
- AI Chatbot:Automated guest support 
- Full-Stack Architecture:Built with PHP, JavaScript, and Laravel Herd.
- Database Management:High-performance database design using DBngin and TablePlus.

Security Implementations (SOC-Focused)
- SQL Injection Prevention: Implemented prepared statements to protect sensitive user data.
- Secure Authentication: Robust protocols for user login and session management.
- Data Normalization: Ensured database integrity and minimized redundancy.
